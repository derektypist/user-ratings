# User Ratings

Create user ratings scorecard with CSS

## UX

As a user, I want to view the user rating based on the number of reviews.

## Features

Allows the user to view the user rating based on the number of views.

## Technologies

Uses HTML5, CSS3 and Font Awesome 7.0.1
